*About Kanabun for Android / かなぶん for Android とは

This application is an Android version of Hiranaga application.
Please refer the web site below about Kanabun itself.
http://kanabun.champierre.com/

ひらがなゲーム「かなぶん」のAndroid版です。
かなぶんについては以下を参照して下さい。
http://kanabun.champierre.com/
